- cd functions-...

- yarn install

- cp contract artifact

- update turkey-2023.js with contract address

- yarn dev

- visit http://localhost:3000 to make a bet

- cd functions..

- make DON request

- fullfill (owner press button Declare via DON)

- make another request in 5 mitnutes
